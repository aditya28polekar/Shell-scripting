Shell Scripting Learning (Part 1)

1. How to create a file?

To create a file in Linux, you can use the touch command. Here's an example:

touch filename.txt

2. Difference between ksh, bash, and echo command in Linux

ksh: Refers to the KornShell, which is a Unix shell that is backward-compatible with the Bourne shell (sh).

bash: Refers to the Bourne-Again Shell, the most common shell used in Linux, offering advanced features and improved scripting capabilities.

echo: A command used to display text or variables on the terminal.

3. How to list files and folders?

To list files and folders in Linux, you can use the ls command. Here are some examples:

# List files and folders in the current directory
ls

# List files and folders with detailed information
ls -l

# List all files and folders, including hidden ones
ls -a

# List files and folders sorted by modification time
ls -lt

4. man command in Linux

The man command is used to display the manual pages for various commands and system functions in Linux. It provides detailed information, usage examples, and command options. Here's an example:

man ls

5. How to write a file in Linux using vi/vim

To write a file in Linux using the vi or vim text editor, follow these steps:

# Open the file in the vi editor
vi filename.txt

# Press 'i' to enter insert mode and start writing
This is some text.

# Press 'Esc' to exit insert mode
# Type ':wq' to save and exit the file
:wq

6. Difference between touch and vim

touch is used to update the access and modification timestamps of a file or create an empty file if it doesn't exist. vim is a powerful text editor that allows you to create, edit, and modify files.

7. How to copy content in Linux?

To copy content in Linux, you can use the cp command. Here's an example:

# Copy a file
cp source_file.txt destination_file.txt

# Copy a directory and its contents recursively
cp -r source_directory/ destination_directory/

8. Purpose of #!/bin/bash or #!/bin/sh in a script

The #!/bin/bash or #!/bin/sh line, also known as a shebang, specifies the interpreter (shell) that should be used to execute the script. In this case, /bin/bash or /bin/sh is the path to the Bash or Bourne shell executable, respectively.

9. How to use the insert command in Linux?

The insert command is not a standard Linux command. If you meant the insert mode in text editors like vi or vim, you can enter insert mode by pressing the 'i' key. In insert mode, you can type and edit text. Press 'Esc' to exit insert mode.

10. How to execute a Shell Script?

To execute a shell script in Linux, follow these steps:

# Make the script executable (if it's not already)
chmod +x script.sh

# Run the script
./script.sh

11. How to grant permissions in Linux? chmod command and how to use it?

The chmod command is used to change permissions (read, write, execute) of files and directories in Linux. Here's an example:

# Grant read, write, and execute permissions to the owner
chmod u+rwx filename.txt

# Grant read and execute permissions to the group
chmod g+rx filename.txt

# Grant read permissions to others
chmod o+r filename.txt

12. How to check the history of commands?

To check the history of commands executed in a shell session, you can use the history command. Here's an example:

history

13. How to create folders?

To create folders (directories) in Linux, you can use the mkdir command. Here's an example:

mkdir foldername

14. How to change the directory in Linux?

To change the current working directory in Linux, you can use the cd command. Here's an example:

cd /path/to/directory

15. How to write a simple shell script?

To write a simple shell script, create a new file with a .sh extension and add your commands to it. Here's an example:

#!/bin/bash

# This is a simple shell script
echo "Hello, World!"

16. What is the purpose of Shell Scripting in DevOps?

Shell scripting plays a vital role in DevOps by automating repetitive tasks, facilitating system administration, and enabling the deployment and management of applications. It allows for the orchestration of various DevOps processes, such as provisioning infrastructure, configuring environments, deploying applications, and managing system resources.

17. How to check CPU and RAM of a Linux Machine?

To check CPU and RAM information in Linux, you can use various commands:

  • top: Provides real-time information about CPU usage, memory usage, and other system statistics.
  • htop: Similar to top, but with an interactive and user-friendly interface.
  • free: Displays information about memory usage and available memory.
  • cat /proc/cpuinfo: Shows detailed information about the CPU(s) in the system.

18. What is the "top" command and why is it used?

The top command is used to monitor system resources in real-time, providing a dynamic view of processes, CPU usage, memory usage, and other system statistics. It is commonly used for performance monitoring, troubleshooting, and identifying resource-intensive processes.

19. Conclusion

Shell scripting is a powerful tool for automating tasks, managing systems, and enabling DevOps practices. It allows you to create scripts to perform repetitive tasks, manipulate files and directories, manage permissions, and interact with system resources. Learning shell scripting can greatly enhance your efficiency and productivity as a Linux user or a DevOps practitioner.
